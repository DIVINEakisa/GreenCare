## 🎨 Typography

### Headings
- **H1**: 3rem, bold, green color
- **H2**: 2.2rem, bold, green color
- **H3**: 1.5rem, semi-bold, dark color
- **H4**: 1.1rem, semi-bold, green color

### Body Text
- **Regular**: 1rem, light gray
- **Secondary**: 0.9rem, lighter gray
- **Large**: 1.1rem, dark gray

### Font
- **Font Family**: Segoe UI, Tahoma, Verdana
- **Line Height**: 1.6 (spacious)
- **Letter Spacing**: Minimal

## 📱 Responsive Behavior

### Large Desktop (>1200px)
- Full grid layouts
- Side-by-side content
- Wide spacing

### Desktop (769px - 1200px)
- 2-3 column layouts
- Medium spacing
- Optimized for readability

### Tablet (481px - 768px)
- 2 column layouts
- Adjusted spacing
- Touch-friendly

### Mobile (<480px)
- Single column
- Vertical stacking
- Large touch targets
- Simplified layouts

## 🌟 Visual Hierarchy

### Importance Order
1. **Hero Section** (Biggest, most prominent)
2. **Headings** (Large, green, bold)
3. **Buttons** (Bright green, interactive)
4. **Cards** (Content containers)
5. **Text** (Body content)
6. **Icons** (Visual aids)
7. **Footer** (Navigation only)
